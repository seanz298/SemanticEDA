# SemanticEDA Roadmap

---

# Vision

SemanticEDA explores AI-assisted semantic electronic design.

The project focuses on:

- system-level understanding
- electronic semantics
- engineering-aware PCB generation
- AI-assisted hardware architecture

---

# Phase 1 — Semantic Understanding

## V0.1 — Natural Language Parsing

Goals:

- parse hardware intent
- extract modules
- identify constraints
- generate structured design descriptions

---

## V0.2 — Electronic Knowledge Graph

Goals:

- component relationships
- support circuit understanding
- bus topology understanding
- power architecture reasoning

---

## V0.3 — Component Recommendation

Planned integrations:

- LCSC
- DigiKey
- Mouser

Goals:

- package selection
- stock awareness
- price awareness
- engineering compatibility
